
<h1 align="center">vitepress-theme-vuetom docs 示例</h1>

## 快速开始

* 安装依赖

```bash
pnpm install
```

* 启动博客示例

```bash
# 本地开发
pnpm dev

# 打包
pnpm build

# 预览
pnpm serve
```

## 主题修改

自 vitepress-theme-vuetom v2.2.x 开始以将主题放置在根目录中

修改根目录下 vuetom 主题文件即可
