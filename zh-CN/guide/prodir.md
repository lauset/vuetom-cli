# 主题目录

在使用一个框架，其实也要简单了解下该框架的项目文件结构，请向下看吧。

```bash
vuetom
  ├─ components        主题Vue组件      文件夹
  ├─ composables       组件脚本         文件夹
  ├─ layouts           布局组件         文件夹
  ├─ styles            全局样式         文件夹
  ├─ support           供支持脚本       文件夹
  ├─ types             ts规范描述       文件夹
  ├─ vuetomui          内置UI库         文件夹
  ├─ config.js         主题配置         文件
  ├─ constant.js       常量定义         文件
  ├─ index.js          主题入口文件      文件
  └─ utils             工具类           文件
```

接下来介绍本文档**docs**文件夹

```bash
docs
  ├─ .vitepress
  │   └─ config.js   网络API接口文件
  │ 
  ├─ public          静态文件资源
  │ 
  ├─ zh-CN           中文菜单
  │    ├─ feedback   留言反馈
  │    ├─ guide      指引
  │    ├─ mdshow     Markdown示例
  │    ├─ menu       UI组件
  │    ├─ styl       主题与样式
  │    └─ index.md   中文首页
  │ 
  ├─ crowdin.yml      多国语言配置
  ├─ index.md         项目首页
  ├─ package.json     包配置
  ├─ README.md        项目说明
  ├─ tsconfig.json    ts配置
  └─ vite.config.ts   vite配置
```
